# JavaBackend

This repository contains exercises and projects developed during my studies in backend development using Java. The main focus is learning and practicing Java, covering fundamental to advanced topics.

## Repository Structure

- `src/`: Contains source code for exercises organized by modules and packages.
- `.gitignore`: Specifies which files or folders should be ignored by Git.
- `README.md`: Repository documentation.

## Technologies Used

- **Java**: The programming language used in the exercises.

## How to Run the Exercises

### Clone the Repository:
```bash
git clone https://github.com/FernandoRiggi/JavaBackend.git
```

### Navigate to the project directory:
```bash
cd JavaBackend
```

### Compile and Run:
```bash
javac src/FileName.java
java -cp src FileName
```
